# SAS Validate - Market Validation Platform

AI-powered market validation and research platform by KoreOps AI.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- TanStack Query

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```

## Deployment

Frontend is deployed on Vercel at: https://market-insights-hub-48.vercel.app/

Backend API is deployed at: https://sas-api-two.vercel.app/
